# DictSwitcher (讀音字庫切換)

這是一個專為 NVDA (NonVisual Desktop Access) 螢幕閱讀器設計的附加元件，旨在幫助使用者在不同的閱讀情境下，快速切換多個自訂的讀音字典。

## 功能特色

* 
**多字典支援**：可同時載入多個 `.dic` 字典檔，例如一般修正、數學專業術語或點字專用讀音 。


* **快捷鍵循環切換**：按下 `NVDA + Alt + D` 即可在「一般」、「數學」、「點字」及「停用」之間循環切換。
* **語音即時回饋**：切換時 NVDA 會朗讀當前使用的字典名稱，確保使用者清楚目前的設定。
* **安全保護機制**：針對 NVDA 2025 新版架構開發，內建錯誤處理程序，確保即便字典檔案異常也不會導致 NVDA 失去語音輸出。

## 檔案結構

本附加元件包含以下核心檔案 ：

* 
**`manifest.ini`**：附加元件的設定資訊與版本說明 。


* 
**`globalPlugins/`**：存放核心程式碼與字典檔的資料夾 。


* 
**`my_dictionary_plugin.py`**：處理字典切換邏輯的 Python 腳本 。


* 
**`my_dict.dic`**：存放一般讀音修正的字典檔 。


* 
**`math_dict.dic`**：存放數學符號相關讀音的字典檔 。


* 
**`brl_dict.dic`**：存放點字情境相關讀音的字典檔 。





## 使用方式

1. 
**安裝**：下載並安裝 `DictSwitcher.nvda-addon` 。


2. **切換字典**：按下快捷鍵 **`NVDA + Alt + D`**。
3. **字典循環順序**：
* 使用：一般讀音修正
* 使用：注音點字字庫
* 使用：數學點字字庫
* 停用自訂字典



## 自訂字典內容

如果你需要新增或修改讀音規則，請編輯 `globalPlugins/` 資料夾下的 `.dic` 檔案。

> **注意：** 編輯時請確保檔案第一行為 `nvda_speech_dict_v1`，且儲存編碼為 **UTF-8 (不帶 BOM)**。
